# Notes Saver v2

A menu-driven Notes Saver application built in Python.

## Features

- Add notes
- View saved notes
- Save notes in a text file
- Exit the application

## Concepts Used

- Python
- While Loop
- If-Elif-Else
- User Input
- File Handling
- with open()
- read()
- write()

## How to Run

Run:

```bash
python notes_saver_v2.py
```

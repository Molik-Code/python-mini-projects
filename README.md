# Student Report System
## Description

This is a Python mini project that analyzes the marks of five students.

The program calculates:

- Highest Marks
- Lowest Marks
- Average Marks
- Pass Count
- Fail Count
- Grades

## Concepts Used

- Variables
- Input / Output
- Lists
- Functions
- for Loop
- if-elif-else
- return

## Grading System

| Marks | Grade |
|-------|-------|
| 90+ | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| 40-59 | E |
| Below 40 | Fail |

## Restrictions

This project was built without using:

- max()
- min()
- sum()
- len()

## Example

Input:

95
82
76
65
30

Output:

Highest: 95
Lowest: 30
Average: 69.6
Pass Count: 4
Fail Count: 1
Grades:
['A', 'B', 'C', 'D', 'Fail']
